# 算法竞赛知识点整理

这是一个用于本人整理算法竞赛相关知识点和解题思路的仓库。

## 目录结构

- `基础算法/` - 包含基础算法知识点和实现
- `数据结构/` - 包含各种数据结构的实现和应用
- `动态规划/` - 包含动态规划相关题目和解题思路
- `图论/` - 包含图论相关算法和题目
- `数学/` - 包含数论、组合数学等数学相关知识点
- `模板/` - 包含常用算法模板代码

## 使用说明

1. 每个知识点目录下包含：
   - 知识点讲解文档
   - 相关例题
   - 解题思路
   - 代码实现

2. 建议按照以下方式使用：
   - 先阅读知识点讲解
   - 尝试解决例题
   - 参考解题思路
   - 学习代码实现

## 贡献指南

欢迎提交PR来完善这个知识库。提交时请注意：
1. 保持代码风格统一
2. 添加必要的注释
3. 确保代码正确性
4. 更新相关文档
